Project 1: LLM-based Macau Pattern Styles Learning and Generation

A project focused on learning and generating Macau pattern styles using large language models, integrating cultural datasets and design analysis.

This project explores the intersection of artificial intelligence and cultural design, specifically focusing on Macau patterns. The main components include:

  • a. Establishing a Dataset of Cultural and Creative Product Images: Collect examples of excellent cultural and creative products through web scraping, focusing on products related to Dunhuang and the Palace Museum (e.g., calendars, cups, etc.)
  • b. Conducting Design Analysis with Large-Scale Models: Perform pattern framework analysis, conduct color analysis, and explore other design-related features using advanced AI models
  • c. Integrating Macau Pattern Dataset: Use the Macau pattern dataset created in step (a) and apply large-scale language models to generate design results, combining Macau patterns with other cultural and creative products
  • d. Developing an Application: Build an app to help designers implement the above functions, providing tools for dataset usage, design analysis, and creative generation
